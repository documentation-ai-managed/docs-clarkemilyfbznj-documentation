## Upgrading Your Installation

You upgrade Clarkemilyfbznj Documentation seamlessly across package managers. Follow these steps to get the latest version and apply any migrations.

<Tabs>
  <Tab title="npm" icon="package">
    <Steps>
      <Step title="Update Package" icon="download">
        Run the update command in your project root.

        <CodeGroup tabs="Terminal">
          ```bash
          npm install clarkemilyfbznj-docs@latest
          ```
        </CodeGroup>
      </Step>
      <Step title="Run Migrations" icon="settings">
        Execute the migration script if upgrading from v1.0.

        ```bash
        npx clarkemilyfbznj-docs migrate
        ```
      </Step>
      <Step title="Restart Server" icon="refresh-cw">
        Restart your development server.

        ```bash
        npm run dev
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="Yarn" icon="package">
    <Steps>
      <Step title="Update Package" icon="download">
        ```bash
        yarn add clarkemilyfbznj-docs@latest
        ```
      </Step>
      <Step title="Run Migrations" icon="settings">
        ```bash
        yarn clarkemilyfbznj-docs migrate
        ```
      </Step>
      <Step title="Restart Server" icon="refresh-cw">
        ```bash
        yarn dev
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Callout kind="tip">
  Back up your `docs/` folder before upgrading. Review migration notes in the release details above for version-specific changes.
</Callout>
